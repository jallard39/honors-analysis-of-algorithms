Problem 1

Input specification: the first line contains n and m (the number of vertices and edges of the graph). The vertices are labeled with numbers 1, 2, ..., n. The first line is followed by m lines, each of these lines describes an edge. The i-th of these lines contains four numbers. The first two numbers specify the end-point vertices, the third number is the weight and the fourth number is either 0 or 1 (1 means the edge belongs to F, 0 means the edge is not in F). You may assume that n is not bigger than 10000 and the weights fit in double.

Output specification: the output contains one line with either the cost of a minimum-cost F-containing spanning tree, or -1 if no such spanning tree exists.

Sample input: input-1.1, input-1.2, input-1.3, input-1.4, input-1.5
Sample output: answer-1.1, answer-1.2, answer-1.3, answer-1.4, answer-1.5


How to submit: Name your file SpanTree.java or SpanTree.cpp


Problem 2

Input specification: the first line contains n and m, the number of vertices and edges. The vertices are labeled with numbers 1, 2,..., n. Then m lines follow. The i-th line contains u, v, and w, specifying the i-th edge that goes from vertex u to vertex v, of weight w. You may assume that n is at most 10000 and that the weights fit in int.

Output specification: the first line contains the string "YES" if there exists a negative-weight cycle, otherwise it contains the string "NO".

Sample input: input-2.1, input-2.2, input-2.3, input-2.4, input-2.5, input-2.7
Sample output: answer-2.1, answer-2.2, answer-2.3, answer-2.4, answer-2.5, answer-2.7

How to submit: Name your file NegativeCycle.java or NegativeCycle.cpp


Problem 3

Input specification: the first line contains n, a, b, and c. We number the hats 1,...,a, the mitten pairs 1,...,b, and the jackets 1,...c. Then 3n lines follow. The i-th group of three lines specifies the preferences of the i-th child. The first of these lines specifies which hats the child finds acceptable: it contains a sequence of distinct numbers, all between 1,...,a, ended with the number 0. The second and third lines similarly specify the child's preferences for mitten pairs and for jackets, respectively. You may assume that n, a, b, and c are each at most 100.

Output specification: the first line contains the string "YES" if it is possible to find an acceptable hat, pair of mittens, and a jacket simultaneously for all children; otherwise the string is "NO". If the string is "YES", then n lines follow. The i-th line lists the hat, mitten pair, and jacket for the i-th child. (Notice that this part of the output is not necessarily unique. All correct outputs will be accepted for full credit.)

Sample input: input-2.1, input-2.2, input-2.3, input-2.4
Sample output: answer-2.1, answer-2.2, answer-2.3, answer-2.4

How to submit: Name your file Daycare.java or Daycare.cpp
