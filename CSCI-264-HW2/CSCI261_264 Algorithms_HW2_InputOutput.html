Problem 1

Input specification: The input consists of several lines. The first line contains a positive integer n. Then n lines follow. The i-th line contains coordinates xi, yi, separated by a space. You may assume that the coordinates are integers that fit in int. You may also assume that n is not larger than 1,000,000.

Output specification: The output contains a single line with the maximum possible number of crossings.

Sample inputs: input-1.1, input-1.2, input-1.3, input-1.4, input-1.5, input-1.6, input-1.7, input-1.8
Sample outputs: answer-1.1, answer-1.2, answer-1.3, answer-1.4, answer-1.5, answer-1.6, answer-1.7, answer-1.8

How to submit: Name your files HorizontalCrossing.java or HorizontalCrossing.cpp

Problem 3

Input specification: The input consists of two lines. The first line contains a positive integer n. The next line contains n non-negative integers b1, ..., bn. You may assume that n is not larger than 1,000,000 and that all other input integers fit in int.

Output specification: The output contains a single line with the weighted count of inversions for the input sequence.

Sample input: input-3.1, input-3.2, input-3.3, input-3.4, input-3.5, input-3.6, input-3.7
Sample output: answer-3.1, answer-3.2, answer-3.3, answer-3.4, answer-3.5, answer-3.6, answer-3.7

How to submit: Name your files WeightedBySumInversions.java or WeightedBySumInversions.cpp
