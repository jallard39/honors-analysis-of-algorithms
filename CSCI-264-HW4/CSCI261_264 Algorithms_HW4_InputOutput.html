Input/output specifications (before reading this, read the problem statements!!!):


Problem 1

Input specification: The first line contains n and p. The next line contains a1, ..., an, separated by spaces. You may assume that all the numbers are positive, their sum fits in int and that n is at most 10,000.

Output specification: The output consists of a single line, containing the maximum sum of an increasing subsequence of a1, ..., an with gap at most p.

Sample input: input-1.1, input-1.2, input-1.3, input-1.4, input-1.5, input-1.6, input-1.7, input-1.8
Sample output: answer-1.1, answer-1.2, answer-1.3, answer-1.4, answer-1.5, answer-1.6, answer-1.7, answer-1.8

How to submit: Name your file IncrSubseqGap.java or IncrSubseqGap.cpp


Problem 2

Input specification: the first line contains n, the number of intervals. Then n lines follow. The i-th line contains two numbers si and fi indicating the start and the finish time of the i-th interval. Then another n lines follow: the i-th of these lines contains n numbers, the j-th of them indicates bi,j. You may assume that n is not bigger than 10,000 and all the input numbers fit in int.

Output specification: the output contains a single line with the maximum number of non-overlapping intervals that satisfy the break requirement - between any pair of chronologically consecutive selected intervals i and j, there has to be a gap of at least bi,j time.

Sample input: input-1.1, input-1.2, input-1.3, input-1.4, input-1.5, input-1.6 (very large; sorry!)
Sample output: answer-1.1, answer-1.2, answer-1.3, answer-1.4, answer-1.5, answer-1.6

How to submit: Name your file IntervalsBreaks.java or IntervalsBreaks.cpp


Problem 3

Input specification: the first line contains n, W1, and W2, this is followed by n lines where the i-th line describes the i-th item: it contains wi and ci. You may assume that n is not bigger than 100, W1 and W2 are not bigger than 1,000 and that all weights and costs are positive numbers that fit in int.

Output specification: the output contains three lines. The first line contains the maximum cost of a set of items that fits in the two backpacks. The second line lists the items to include in the first backpack and the third line lists the items to include in the second backpack. The items are listed in increasing order of their IDs and are separated by spaces.

(Note: the output is not necessarily unique. Print any of the solutions.)

Sample input: input-4.1, input-4.2, input-4.3, input-4.4, input-4.5
Sample output: answer-4.1, answer-4.2, answer-4.3, answer-4.4, answer-4.5

How to submit: Name your file DoubleKnapsackWithSolution.java or DoubleKnapsackWithSolution.cpp


Problem 4

Input specification: the first line contains n and the next line contains a0, ..., an, separated by spaces. You may assume that all the numbers are positive and fit in int and that n is at most 1,000.

Output specification: The output contains two lines. The first line is the optimal multiplication cost. The second line contains an optimal parenthesizing printed in the following form:

  • For a matrix, print Ai, where i is the number of the matrix.
  • For the multiplication of two matrices B and C, print B x C, with spaces between the matrices and the multiplication sign. Notice that B and/or C might be parenthesized expressions.
For example: ( ( A1 x ( A2 x A3 ) ) x ( A4 x A5 ) )
(Note: the output is not necessarily unique. Print any of the solutions.)

Sample input: input-3.1, input-3.2, input-3.3, input-3.4, input-3.5
Sample output: answer-3.1, answer-3.2, answer-3.3, answer-3.4, answer-3.5

How to submit: Name your file MatrixChainParenthesize.java or MatrixChainParenthesize.cpp


Problem 5

Input specification: the first line contains n and the next line contains a1, ..., an, separated by spaces. You may assume that all the numbers fit in int and that n is at most 1,000.

Output specification: the output contains one line with the maximum length of a convex subsequence of a1, ..., an.

Sample input: input-1.1, input-1.2, input-1.3, input-1.4, input-1.5
Sample output: answer-1.1, answer-1.2, answer-1.3, answer-1.4, answer-1.5

How to submit: Name your file LongestConvexSubseq.java or LongestConvexSubseq.cpp
