Input/output specifications (before reading this, read the problem statements!!!):


Problem 1

Input specification: the first line contains n. Following that are n lines of data. The i-th of these lines contains two integers, xi and yi, separated by a space, representing the x and y coordinates of the i-th police car. You may assume that n is not bigger than 10,000 and that all the coordinates are integers with absolute value no greater than 10,000.

Output specification: the output contains one line with the minimum possible sum of distances to every police car (notice, you will output the sum of the distances, not the best location for the donut store).

Sample input: input-1.1, input-1.2
Sample output: answer-1.1, answer-1.2

How to submit: Name your file Donut.java or Donut.cpp


Problem 3

Input specification: the first line contains n and the next line contains a1, ..., an, separated by spaces. You may assume that all the numbers fit in int.

Output specification: the output contains one line with the maximum length of an increasing subsequence of a1, ..., an.

Sample input: input-2.1, input-2.2, input-2.3, input-2.4, input-2.5
Sample output: answer-2.1, answer-2.2, answer-2.3, answer-2.4, answer-2.5

Note: No upper-bound on n is specified. Run both the recursive and the dynamic programming algorithm on a variety of inputs, increasing n, and report on or plot the running times.

How to submit: Name your files LongestIncreasingSubseqRecursive.java and LongestIncreasingSubseqDP.java, or LongestIncreasingSubseqRecursive.cpp and LongestIncreasingSubseqDP.cpp


Problem 4

Input specification: the first line contains n and the next line contains a1, ..., an, separated by spaces. You may assume that n between 1 and 1,000,000 and that the other numbers are integers between 1 and 1000.

Output specification: the output contains one line with the maximum score of this hopscotch game.

Sample input: input-5.1, input-5.2, input-5.3, input-5.4, input-5.5, input-5.6, input-5.7, input-5.8
Sample output: answer-5.1, answer-5.2, answer-5.3, answer-5.4, answer-5.5, answer-5.6, answer-5.7, answer-5.8

How to submit: Name your file Hopscotch.java or Hopscotch.cpp
